# Nikoloz Ninua Portfolio

Welcome to my game design and audio specialist portfolio! This site showcases my work as a game designer, audio specialist, and QA tester from my time at FutureGames.

## Features

- **Responsive Design** - Works beautifully on all devices
- **Bold Brand Identity** - Custom color palette with orange and deep blue accents
- **Smooth Animations** - Premium feel with thoughtful interactions
- **Accessibility** - Built with semantic HTML and keyboard navigation in mind
- **Project Showcase** - Links to my itch.io games with live screenshots

## Projects Featured

- **Catsmith** (Unreal Engine) - Co-op crafting game with forging mechanics
- **Nana Into The Banana** (Unity) - Fast-paced mobile combat game
- **To Bee Or Not To Bee** (Unity) - Eco-friendly platformer with full audio design

## Tech Stack

- HTML5
- CSS3 (Custom styling, no frameworks)
- Vanilla JavaScript
- GitHub Pages hosting
